  • assertBridgeBufferNeverOverflows()

    Raises a fatal error when an upstream RxBridge publisher’s buffer overflows and otherwise maps the failure type to the Error type of an upstream Observable.

    Use this operator at some point following the RxBridge operator if you can be sure that a buffer overflow will never occur.
